# 政策周期模块 (Policy Cycle Impact)

在 A 股市场，**"政策即基本面"**。政策不仅影响行业天花板，更直接决定了资金的风险偏好（估值溢价）。所有的行业研究必须穿透到国家宏观政策层面。

## 1. 政策方向定调 (Policy Stance)
研报必须明确指出研究对象所处行业的国家政策定调，分为三类：
- **🟢 支持/鼓励（如新质生产力、信创、国产替代、出海）**：享受高估值溢价，资金容忍度高，补贴或税收优惠实质落地。
- **🟡 中性/规范（如传统制造、一般消费）**：跟随宏观经济周期波动，无明显政策红利。
- **🔴 压制/出清（如教培(历史)、部分房地产、高能耗产能）**：面临估值折价，逻辑被破坏，任何利好都容易被资金当做出货机会。

## 2. 政策影响传导路径 (Impact Transmission)
政策不只是口号，必须分析政策如何传导至财务报表或竞争格局：
- **影响收入（顶层订单）**：如十四五军工采购规划、车购税减免、设备更新补贴。
- **影响成本（供给侧改革）**：如环保限产导致上游原材料涨价，碳排放配额成本。
- **影响竞争格局（牌照与准入）**：如创新药医保目录谈判、集采规则变化、反垄断罚款。

## 3. 政策周期节点预判 (Policy Timeline)
当前处于政策周期的哪个阶段？
1. **吹风期（政策底）**：高层会议提及，细则未出。（股价开始炒预期，拔估值）
2. **落地期（细则出台）**：部委发布具体执行方案和补贴金额。（资金开始兑现，分歧加大）
3. **业绩期（财报体现）**：政策红利体现在利润表。（真正能走长牛的企业开始脱颖而出）
4. **退出期（补贴退坡）**：如新能源车补贴退出。（估值杀跌，行业洗牌）

## 4. 输出模板范例
```markdown
### 政策周期与监管环境
- **当前政策定调**：[支持 / 中性 / 压制]
- **核心催化文件/事件**：[文件名称或会议精神]
- **周期所处阶段**：[吹风期 / 落地期 / 业绩期 / 退出期]
- **对基本面的实质影响**：该政策预计将在 [时间维度] 内，通过 [传导路径，如提升毛利/扩大需求/加剧竞争] 对公司的 [具体财务指标] 产生实质性影响。
```

## 🧠 核心思想 (Core Ideology)

绝大多数 AI 炒股助手失败的原因，在于它们机械地套用美股“巴菲特式”的价值投资逻辑。然而，在高度博弈的中国 A 股市场，纯粹的财务分析是极其脆弱的。

本智能体的核心思想建立在 A 股买方实战的第一性原理之上：
> **“基本面决定底部，资金面决定高度，政策面决定生死。”**

为此，我们构建了以下 **四大核心框架**，彻底重塑 AI 投研标准：

### 1. 📜 研究宪法 (Research Constitution)
作为防幻觉的最高铁律，AI 在运行本技能时被刻入了 **“七大不妥协纪律”**：
- **禁止无端猜测**：没有数据必须回答“证据不足”，UNKNOWN 永远优于伪造数据。
- **数据溯源**：所有核心财务数据、倍数必须附带来源。
- **事实与推演隔离**：严格区分“既定事实（财报数据）”、“第三方证据”、“逻辑推演”与“主观观点”。
- **预测证伪机制**：任何对未来的预测必须强制说明：①依据 ②假设条件 ③证伪条件。
- **冲突不平均**：遇到券商之间的数据冲突，禁止取平均值，必须说明采信哪一方及其理由。
- **双重孤证**：每个极度重要的结论必须至少拥有两个独立的来源。
- **寻找反例**：每个看多理由必须配套一条可能的反驳逻辑。

### 2. 🇨🇳 中国市场适配层 (China Equity Adaptation)
彻底抛弃真空环境下的纯基本面阅读，强制加载三大 A 股特化引擎：
- **资金博弈 (Capital Flow)**：分析当前定价权归属（外资 / 公募抱团 / 游资接力 / 国家队护盘），以及筹码结构（大股东减持、高换手率、解禁压力）。
- **政策周期 (Policy Cycle)**：精准定调行业当前面临的国家政策是“支持”、“中性”还是“压制”；目前处于炒估值的“吹风期”、业绩兑现的“落地期”还是杀估值的“退潮期”。
- **证据等级评级体系**：从 5星（官方财报/公告）到 1星（雪球/股吧情绪），智能体会根据来源给予信息不同的采信权重。

### 3. 🏭 工业级十二步流水线 (12-Step SOP)
告别 AI 的“随意聊天”，强制执行从接到任务到做出决策的标准化买方工序，**禁止跳步**：
1. 理解需求与目标。
2. 拆解多维研究树。
3. 覆盖式地毯搜索（财报+政策+资金+对手）。
4. **证据入账**（构建 XML 记忆底座）。
5. 政策与产业链定位。
6. 资金面扫描与筹码博弈。
7. **预期差分析**（市场一致预期 vs 真实基本面）。
8. 蓝军建构（≥5条看多硬逻辑）。
9. 红军建构（≥5条致命看空逻辑）。
10. **红队攻击自检**。
11. **买方实战决策**（左侧布局 / 右侧跟随 / 观望等待 / 坚决规避）。
12. Firewall 交付检查。

### 4. 🛡️ 防幻觉与红队攻击 (Anti-Hallucination & Red Team)
- **`<evidence_ledger>` 记忆锁**：在长文本生成前，强制在一个封闭的 XML 标签内吐出所有核心数字，防止大模型在生成万字报告时发生“注意力遗忘”或“数字变异”。
- **红队自我攻击**：在给出投资结论后，系统必须自我审视：“我这份报告最大的事实错误可能在哪里？逻辑漏洞是什么？最可能打脸我的事件是什么？”——确保在狂热情绪中保持极端清醒。
- **Firewall 强制打印**：在最终报告结尾强制打印自检清单，但凡缺少“多空交锋”或“最新财报”，直接推倒重写。

## 🛠️ 衍生功能亮点 (Feature Highlights)

除了深度的定性研报生成，本智能体还具备以下绝活：

- **Mermaid 产业链全景图**：强制调用 Markdown Mermaid 引擎。在每份报告中，自动渲染出清晰的“上下游产业链流转节点”与“核心利润流向”图谱。
- **A_Share_Quant_Scorer 铁血裁判**：自带的一个专为散户设计的量化打分微型智能体。能够根据牛熊市动态调整因子权重，给出 0-100 的六维雷达评分，并遇到“巨额解禁”或“流动性枯竭”时，坚决执行一票否决清零！

## 🚀 快速开始

### 安装指南
针对使用 Antigravity 或类似 AI 终端框架的用户，直接将本仓库克隆至全局定制化目录即可生效：

```bash
mkdir -p ~/.gemini/config/skills
cd ~/.gemini/config/skills/
git clone https://github.com/nihong/company-researcher.git
```

### 触发方式
安装完成后，在终端直接向您的 AI 助手输入自然语言即可极速唤醒：

```text
"帮我调研一下兴福电子，并给出量化打分。"
"以买方视角分析海光信息的投资机会。"
"对比茅台和五粮液当前的资金筹码结构与政策周期。"
```

> **💡 多 Agent 协同玩法 (Teamwork Preview)**: 
> 结合底层多模型调度（如 Claude 3.5 + DeepSeek + Gemini），您可以让本智能体指挥多名虚拟研究员并行搜集数据，并在汇总交叉验证后，直接为您生成精美的 PDF 并发送至您的邮箱。

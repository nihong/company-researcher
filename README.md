## 🧠 核心思想 (Core Ideology)

绝大多数 AI 炒股助手失败的原因，在于它们机械地套用美股“巴菲特式”的价值投资逻辑。然而，在高度博弈的中国资本市场，纯粹的财务分析是极其脆弱的。

本智能体的核心思想建立在实战的第一性原理之上：
> **“基本面决定底部，资金面决定高度，政策面决定生死。”**

为此，我们构建了以下 **五大核心框架**，彻底重塑 AI 投研标准：

### 1. 📜 研究宪法 (Research Constitution)
作为防幻觉的最高铁律，AI 在运行本技能时被刻入了 **“七大不妥协纪律”**：
- **禁止无端猜测**：没有数据必须回答“证据不足”，UNKNOWN 永远优于伪造数据。
- **事实与推演隔离**：严格区分“既定事实”、“第三方证据”、“逻辑推演”与“主观观点”。
- **预测证伪机制**：任何预测必须强制说明：①依据 ②假设条件 ③证伪条件。

### 2. 🇨🇳 市场深度适配层 (A/H Share Adaptation)
彻底抛弃真空环境下的纯基本面阅读，强制加载**资金博弈、政策周期、产业链景气度**三大外围约束引擎：
- **A 股特化**：公募抱团追踪、北向资金、龙虎榜游资情绪、大小非解禁压力、T+1交易约束。
- **港股特化 (v3.1新增)**：智能识别 `.HK` 后缀，自动切换至**南向资金、高股息溢价、A-H折溢价、做空狙击防范**以及**老千股排雷**逻辑。

### 3. 🤖 量化打分引擎与基金经理风控 (Quant & Risk Control)
告别 AI “和稀泥”式的“建议关注”，内置一个专为散户设计的**铁血裁判智能体**：
- **执行可行性风控 (v4.0新增)**：强迫 AI 面对“T+1跌停无法止损”的物理现实！在给出任何买卖策略前，强制执行包含 T+1 影响、跌停风险的《交易执行可行性矩阵》评估，杜绝纸上谈兵。
- **板块拥挤度惩罚 (v3.5新增)**：深度解决 A 股“存量博弈”死穴。当个股所在板块成交额占两市 > 10% 或情绪高潮时，直接触发“极度拥挤”惩罚，动量得分强制清零，防范散户在热搜高位接盘。
- **自适应估值矩阵 (v3.4新增)**：彻底废弃单一 PE 估值，强制判断行业生命周期：硬科技股用 PEG，成熟红利股用动态股息率与 PB，强周期股强制开启“反向 PE 法则”。
- **强制仓位与盈亏比红线 (v3.3新增)**：哪怕得满分 100 分的 S 级标的，也会被强制戴上【绝对最高仓位限制】（如 B 级绝对封死 5% 上限，严禁越界）与【绝对止损价】的镣铐。
- **精准映射与动态权重**：所有因子硬编码精准映射；根据大盘牛熊状态自动在进攻（重成长/资金）和防守（重质量/估值）权重间切换，消除大模型主观裁量（彻底终结精确性幻觉）。
- **一票否决与财务穿透 (v3.6新增)**：引入更底层的造假雷达。只要碰到“流动性枯竭”、“大股东减持”、“财务粉饰雷（必须列出具体应收占比/现金流金额）”任意一条红线，无视基本面多好，总分强制熔断清零，强行拉黑。

### 4. 🗃️ 强制数据管道与十四步流水线 (Data Pipeline & 14-Step SOP)
禁止大模型凭借幻觉或过时搜索数据写研报：
- **Step 0 强制抓取 (v3.1新增)**：在生成任何一个字前，系统必须强制调用 `opencli eastmoney` 获取实时的 PE/PB、市值、主力资金净流出、十大流通股东，并写入 `<evidence_ledger>` 锁死。后续引用只能从底层账本调取。
- **工业级装配线**：从制定研究树 → 产业链定位（Mermaid出图） → 预期差分析 → 红蓝交锋，14 步工序禁止跳步。

### 5. 🛡️ 七维红队攻击与事后回测 (Red Team & Backtesting)
- **强制七维红队 (v3.1新增)**：报告结尾不仅要自我攻击，且必须覆盖“事实、逻辑、竞争、资金、宏观、估值、执行”七个维度。**每条攻击必须带具体的证伪条件和时间节点**。
- **制度级执行层攻击 (v3.9新增)**：红队攻击的“执行层”被强制重定义为“交易制度与流程障碍”专场（如：T+1跌停导致无法止损、大宗交易价差过大、港股流动性枯竭），与实战痛点 100% 贴合。
- **分级快照追踪 (v3.9新增)**：每份交付的报告都会自动在 `tracking/ledger.csv` 登记快照，且核心字段被设为绝对不可豁免。回测系统可根据**真实收益相关性**来迭代量化因子的权重，让系统“自我进化”。

## 🛠️ 衍生功能亮点 (Feature Highlights)

除了深度的定性研报生成，本智能体还具备以下绝活：
- **📁 结构化知识库归档与自动索引 (v3.7新增)**：自动在私人工作区生成 `Company_Research_Reports` 目录，并采用极客命名法管理报告。系统还会自动维护知识库的全局 `README.md`，为您构建一张清晰的“A股核心资产打分总览表”，点击表格即可直达底层研报。
- **🔄 Delta 增量对比速览 (v3.2新增)**：当您对某家公司发起二次调研时，系统会自动检索历史档案，并在新报告开头生成“Delta Summary”，精准提示您评分发生改变的核心原因，以及前次报告担忧的红队预警是否已被证伪。
- **Mermaid 产业链全景图**：在每份报告中，自动渲染出清晰的“上下游产业链流转节点”与“核心利润流向”图谱。
- **Firewall 强制打印**：在最终报告结尾强制打印自检清单，但凡缺少“数据源”、“多空交锋”或“快照登记”，直接推倒重写。

## 🚀 快速开始

### 安装指南
针对使用 Antigravity 或类似 AI 终端框架的用户，直接将本仓库克隆至全局定制化目录即可生效：

```bash
mkdir -p ~/.gemini/config/skills
cd ~/.gemini/config/skills/
git clone https://github.com/nihong/company-researcher.git
```

> **前置依赖**：为激活核心的数据抓取与量化功能，建议全局安装 `npm i -g opencli`。

### 触发方式
安装完成后，在终端直接向您的 AI 助手输入自然语言即可极速唤醒：

```text
"帮我调研一下航宇科技，并给出量化打分。"
"以买方视角分析腾讯控股（0700.HK）的投资机会。"
"复盘一下上个月我们生成的投研报告，回测一下胜率。"
```

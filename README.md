## 🧠 核心思想 (Core Ideology)

绝大多数 AI 炒股助手失败的原因，在于它们机械地套用美股“巴菲特式”的价值投资逻辑。然而，在高度博弈的中国 A 股市场，纯粹的财务分析是极其脆弱的。

本智能体的核心思想建立在 A 股买方实战的第一性原理之上：
> **“基本面决定底部，资金面决定高度，政策面决定生死。”**

为此，我们构建了 **四大核心框架**：
1. **研究宪法 (Research Constitution)**：防幻觉的最高铁律。没有数据支撑，AI 必须回答“证据不足”，禁止编造。
2. **中国市场适配层 (China Equity Adaptation)**：彻底抛弃真空环境下的财报阅读，将**政策导向**（吹风期/落地期/退潮期）与**筹码博弈**（公募抱团/游资接力/大股东减持）作为权重极高的核心驱动因子。
3. **十二步流水线 (12-Step SOP)**：告别“随意聊天”，强制执行从“目标拆解”到“多空对抗”，再到“防火墙自检”的标准化工业级工序。
4. **红蓝军互搏 (Red Team Attack)**：在给出做多理由后，系统必须自我审视，强制生成等量的致命看空理由与逻辑漏洞，确保在狂热情绪中保持极端清醒。

## 🛠️ 主要功能 (Main Functions)

本智能体主要提供从“定性深度调研”到“定量排雷打分”的端到端投研服务：

### 1. 深度定性研报生成
接收到“调研某公司”或“分析某板块”的指令后，智能体会全自动执行以下工序：
- **政策与景气度定调**：分析标的当前所处的宏观政策周期，及细分行业的景气度传导。
- **资金与筹码解剖**：追踪主力资金动向（龙虎榜/大宗交易）、评估高低位筹码结构。
- **基本面与预期差提取**：不只是罗列财报，而是挖掘“市场一致预期”与“真实基本面”之间的错配（Variant Perception）。
- **实战化投资决策**：拒绝模棱两可的“建议关注”，直接输出具体的买方操作策略（左侧布局 / 右侧跟随 / 观望等待 / 坚决规避）。

### 2. Mermaid 产业链全景可视化
强制调用 Markdown Mermaid 引擎。在每份报告中，自动渲染出清晰的“上下游产业链流转节点”与“核心利润流向”图谱，标的在产业链中的话语权一目了然。

### 3. A_Share_Quant_Scorer 量化排雷裁判（子智能体）
针对散户投资者量身定制的量化打分插件，深度融合 DeepSeek 实战法则：
- **智能大盘感知**：自动识别牛熊市或震荡市，动态切换进攻/防守因子权重。
- **六维雷达评分**：从基本面、景气度、估值、资金、政策、动量六个维度输出 0-100 的量化分数及 ASCII 进度条。
- **一票否决机制**：一旦触发“流动性陷阱（成交额过低）”或“大股东恶意抽血（巨额减持/解禁）”，总分强制清零。

### 4. Firewall 交付自检防火墙
引入 `<evidence_ledger>`（证据账本）死锁关键数据。在最终报告输出前，AI 必须打印自检清单，但凡缺少“多空交锋”或“最新财报”，必须推倒重写。

## 🚀 快速开始

### 安装指南
针对使用 Antigravity 或类似 AI 终端框架的用户，直接将本仓库克隆至全局定制化目录即可生效：

```bash
mkdir -p ~/.gemini/config/skills
cd ~/.gemini/config/skills/
git clone https://github.com/nihong/company-researcher.git
```

### 触发方式
安装完成后，在终端直接向您的 AI 助手输入自然语言即可极速唤醒：

```text
"帮我调研一下兴福电子，并给出量化打分。"
"以买方视角分析海光信息的投资机会。"
"对比茅台和五粮液当前的资金筹码结构与政策周期。"
```

> **💡 多 Agent 协同玩法 (Teamwork Preview)**: 
> 结合底层多模型调度（如 Claude 3.5 + DeepSeek + Gemini），您可以让本智能体指挥多名虚拟研究员并行搜集数据，并在汇总交叉验证后，直接为您生成精美的 PDF 并发送至您的邮箱。

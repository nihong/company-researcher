# 行业自动识别与专项补充规则

AI 必须根据研究对象所属行业，自动在研报分析中补充以下专属分析维度。如果未匹配到下表中的行业，则按通用框架分析，但必须在报告中明确注明"未使用行业专项模板"。

| 行业 | 必须额外覆盖的核心维度 |
|------|-------------|
| **医药/生物科技** | 管线进展 (Pipeline)、临床阶段、FDA/NMPA审批节点、医保谈判降价预期、竞争药物、专利悬崖 |
| **半导体/芯片** | 制程节点、国产化率、ASP趋势、Capex周期、设备/材料国产替代实质进展 |
| **银行/金融** | NIM（净息差）、拨备覆盖率、不良率趋势、资本充足率、房地产/城投敞口 |
| **消费/零售** | 品牌护城河、渠道结构变化（线上/线下）、复购率、同店增长率、库存周转天数 |
| **新能源/汽车** | 渗透率瓶颈、电池技术路线、补贴/关税政策、产能利用率、单车利润 (ASP - 成本) |
| **军工** | 订单交付节奏、装备列装进度、国防预算方向、军贸出口突破 |
| **AI/科技** | 算力需求、模型迭代进度、商业化落地场景、资本开支、竞争壁垒 |
